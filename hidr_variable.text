|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| setting_average      | Setting average                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| iso3                 | ISO3 country code for country-level data (e.g. "IDN" for Indonesia).                                                                                                                                                                                                                                                                                                                                                                                                                              |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| favourable_indicator | 1 for favourable indicators and 0 for non-favourable (adverse) indicators. Favourable indicators measure desirable events that are promoted through public action. For these indicators, the ultimate goal is to achieve a maximum level, either in health intervention coverage or health outcome. Adverse indicators measure undesirable events that are to be reduced or eliminated through public action, where the ultimate goal is to achieve a minimum level.                              |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| indicator_scale      | The scale of the indicator, such as "100" for indicators reported as percentages or "1000" for indicators reported as rates per 1000 population.                                                                                                                                                                                                                                                                                                                                                  |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ordered_dimension    | 1 for ordered dimensions and 0 for non-ordered dimensions or dimensions with two subgroups (i.e. binary dimensions). Ordered dimensions have ordered subgroups that have an inherent positioning and can be ranked (e.g. education or economic status). Non-ordered dimensions have non-ordered subgroups that are not based on criteria that can be logically ranked (e.g. subnationall regions). Binary dimensions compare the situation in two population subgroups (e.g. males and females).  |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| subgroup_order       | This variable indicates the order of subgroups for ordered dimensions. For ordered dimensions (i.e. if ordered_dimension=1), this variable is an increasing sequence of integers starting with the value 1 for the most-disadvantaged subgroup. For non-ordered dimensions and binary dimensions (i.e. if ordered_dimension=0), this variable is 0.                                                                                                                                               |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| reference_subgroup   | 1 indicates the reference subgroup for non-ordered dimensions and binary dimensions. For ordered dimensions (i.e. if ordered_dimension=1), this variable is 0.                                                                                                                                                                                                                                                                                                                                    |
